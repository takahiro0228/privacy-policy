プライバシーポリシー（日本語）

本拡張機能「Salesforce Bulk Field Creator」は、Salesforce 上で項目をまとめて作成する目的だけに使用されます。

本拡張機能が扱う情報は、Salesforce の画面から確認できる項目情報と、Salesforce API を使うために必要な一時的な接続情報です。これらの情報は保存されません。項目作成処理を実行するために必要な範囲にのみ利用します。

取得した情報はすべてブラウザの中だけで処理されます。外部のサーバーへ送られることはありません。第三者に渡されることもありません。

Chrome の保存領域には、拡張機能の動作に必要な設定だけが保存されます。Salesforce から取得した内容は保存しません。

氏名やメールアドレスなどの、個人を特定できる情報は取得しません。広告や分析のために利用されることもありません。

データの送信を伴う処理は存在しないため、ユーザーが操作によってデータ共有を選ぶ必要はありません。拡張機能は常にローカル環境のみで動作します。

お問い合わせは、公開している GitHub Issues または公開メールアドレスをご利用ください。 GitHub Issues: https://github.com/takahiro0228/privacy-policy/issues

Privacy Policy (English)

The browser extension “Salesforce Bulk Field Creator” is used only for creating Salesforce custom fields in bulk.

The extension handles metadata obtained from Salesforce pages and temporary session information required for Salesforce API calls. These data are not stored and are used only for the purpose of executing the field creation process.

All information is processed entirely within the user's browser. No data is transmitted to any external server and no data is shared with any third party.

Only extension settings are stored using Chrome’s built-in storage. No Salesforce data of any kind is saved.

The extension does not collect personally identifiable information such as names or email addresses. No information is used for advertising or analytics.

Because no data is transmitted externally, users do not need to take any action to control data sharing. The extension always operates entirely within the local environment.

For inquiries, please contact through the public GitHub Issues page or the public contact email. GitHub Issues: https://github.com/takahiro0228/privacy-policy/issues
