プライバシーポリシー（日本語）

本拡張機能「Salesforce Bulk Field Creator」は、Salesforce 上で項目を一括作成する目的のみに利用されます。

本拡張機能が扱う情報は次のとおりです。

・Salesforce の画面から取得できるオブジェクト情報と項目情報

・Salesforce API を利用するために必要となるセッション情報

これらの情報はすべてユーザーのブラウザ内で処理されます。

外部サーバーへ送信することはありません。

収集した情報を第三者へ提供することもありません。

Chrome 標準の保存領域（chrome.storage）には、拡張機能の設定情報のみが保存されます。Salesforce から取得したデータは保存されません。

ユーザーの個人情報を広告目的や分析目的で利用することはありません。

お問い合わせは GitHub の Issues または公開メールアドレスをご利用ください。

Privacy Policy (English)

The browser extension “Salesforce Bulk Field Creator” is used solely for the purpose of creating Salesforce custom fields in bulk.

This extension handles the following information only:

・Object and field information obtained from Salesforce pages

・Session information required to use Salesforce APIs

All information is processed within the user's browser.

No data is transmitted to any external server.

No collected information is shared with third parties.

Only extension settings are stored using Chrome’s built-in storage (chrome.storage). No Salesforce data is saved.

The extension does not use any personal information for advertising or analytics.

For inquiries, please contact via GitHub Issues or the publicly available contact email.
